Cross-Platform

Commercial

  • BibleWorkshop 5 (upcoming)

Non-Commercial

Open Source

  • BibleDesktop
    • Footnotes display in a separate column, reducing visual clutter.
  • BPBible (using crossplatform wxPython, may not work on all platforms)

Closed Source

  • Tnach

Windows

Commercial

  • BibleWorkshop
  • Libronix
  • BibleWorks 4.x
  • QuickVerse. Mac version also.
    • PDF Tutorial with screenshots
    • "Essentials" pack, quick to read feature list
    • nice reviews for an older Mac version: http://www.cmug.org/articles/QuickVerse.html, http://www.cmug.org/articles/QuickVerseReview.html
  • Logos 3
    • Video tutorials might be useful for feature planning
    • About the underlying technology
  • WORDsearch
  • SwordSearcher (30-day shareware)

Non-Commercial

Open Source

  • BibleCS
  • SwordBible

Closed Source

  • eSword
    • Probably the most famous free application - watch this closely and make sure BT can do all which this can but better :)
    • Quite clean interface
    • but some clumsyness: popup notes, distracting strong's, some non-intuitive or hard to use features
    • Features:
      • Study notes and Topic notes (i.e. edit your own commentary and dictionary)
      • Bookmarks
      • Highlight texts (but how does it work???)
      • Parallel Gospel Harmony
      • Analyze verses (counts the words in a verse range)
  • OnlineBible
  • The Word
    • v.3 beta1, see especially "What's new" and "Resources".
    • Many features which we want
    • Good example of how to give users freedom and features
    • But very good example of BAD usability design (e.g. 9 visible widgets/buttons/links for Bible navigation, config options for everything - i.e. the developer has not made any good decisions)
  • MyBible
  • Seed Master
  • Free edition of Bible Explorer!
