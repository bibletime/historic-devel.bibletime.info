iTwinkle is our beloved IRC companion. Here are some good tips about how to converse with her:

Listing all plugins

Command: 

iTwinkle: lsmod

Sample output: 

user: Plugins: admin, ascii, calc, conversions, core, eval,
factoid, feeds, film, fun, games, geography, google, help,
icecast, identity, karma, languages, launchpad, log, lotto,
meetings, memo, memory, network, oeis, quotes, rfc, seen, social,
sources, strings, sysadmin, test, urlgrab and urlinfo

Leaving messages

Command: 

iTwinkle: how do I use memo

Sample output: 

user: Keeps messages for people. You can use it like this:
  my messages
  message <number>
  my messages for <person> [on <source>]
  (tell|pm|privmsg|msg|ask) <person> [on <source>] <message>
  forget my (first|last|<n>th) message for <person> [on <source>]

Handling RSS and Atom feeds

Command: 

iTwinkle: how do I use notify

Sample output: 

user: Displays articles from RSS and Atom feeds. You can use it like this:
  add feed <url> as <name>
  remove <name> feed
  list feeds
  poll <name> feed notify <channel> on <source>
  stop polling <name> feed
  latest [ <count> ] articles from <name> [ starting at <number> ]
  article ( <number> | /<pattern>/ ) from <name>
