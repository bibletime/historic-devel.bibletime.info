The meeting on 16 January, 2010, starting from 1900 UTC, was a FEATURE-A-THON.

Features decided on

  • MDI STYLE TOOLBAR FEATURE.
    • Child window toolbars can be either on the child windows or the main window. (e.g. Toolbar A can either be on ALL child windows or ONLY on the main window)
  • MDI WINDOW RESIZE FEATURE.
    • When 2 windows are next to each other with borders side-by-side, resizing these borders will resize the windows next to these borders.
  • DESKTOPS.
    • "Sessions" -> "Desktops"
    • Quick desktop switching toolbar button
    • Desktops save themselves when changed
    • MDI area will not change size when switching desktops
  • MDI TABBED LAYOUT.
    • Implement it (Gary?)

Things to discuss on next meetings

  • Usability issues (especially for first-time users):
    • welcome screen with instructions to install modules (Eeli?)
    • Restructuring the main window menu bar
    • Performance issues
    • Lexicon key chooser (in respect to both usability and performance)
  • Tabs inside windows
